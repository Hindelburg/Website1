About the author.

Prompt generator is a program that uses a series of machine learning methods and a context free grammar to generate the following types of story prompts.

  1. Concept prompts.
  2. Character prompts.
  3. Setting prompts.
  4. Sentence prompts.
  5. Style prompts.
  6. Markov Chains.
